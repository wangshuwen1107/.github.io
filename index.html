#设备设置主页文档描述

##Native Uri
rokid://settings/device/index

业务协议（目前本地写死数据）

Response Json

1.格式

参数名称 类型 说明 列子
type string 业务类型 deviceNick
name string 名称 设备名称
value string 对应名称的值 pebble1ED
description string 描述 开机后，唤醒时会有档的声音
action string 跳转/开关 linked/switch
linkedUrl string action为linked时跳转的链接 rokid://settings/device/nickName
switchAction string action为switch时开关的动作 standbyLight
